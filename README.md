## 🚀 Sobre o Projeto

A **NovaPay** é um projeto de landing page fictício desenvolvido para portfólio, simulando o site institucional de um banco digital moderno. O foco foi criar uma interface limpa, funcional e visualmente impactante usando apenas **HTML, CSS e JavaScript puro** — sem frameworks, sem bibliotecas externas.

<br/>

## ✨ Funcionalidades

- 📱 **Design responsivo** — adaptado para mobile, tablet e desktop
- 🎨 **Identidade visual moderna** — paleta branco + verde, tipografia premium (Syne + DM Sans)
- 🖱️ **Scroll suave** — navegação fluida entre seções ao clicar nos links do menu
- 🔐 **Modal de Cadastro** — com máscara de CPF e celular, toggle de senha e validação de termos
- 🔑 **Modal de Login** — com toggle de senha e troca dinâmica entre modais
- ✅ **Modal de Sucesso** — feedback visual animado após ação do usuário
- 🧲 **Navbar inteligente** — efeito glassmorphism que muda ao fazer scroll
- 📲 **Mockup de celular** — animado, com transações e saldo fictícios
- 💳 **Cards flutuantes** — com animação de levitação
- 🌊 **Scroll reveal** — elementos aparecem suavemente ao entrar na tela
- ☰ **Menu hamburger** — funcional para telas menores

<br/>

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura semântica da página |
| **CSS3** | Estilização, animações, responsividade e modais |
| **JavaScript (ES6+)** | Interações, modais, scroll, máscaras de input |
| **Google Fonts** | Tipografia (Syne + DM Sans) |

> Nenhum framework ou biblioteca JS foi utilizado. Tudo feito na raça. 💪

## 📌 Aprendizados & Destaques Técnicos

- **CSS Custom Properties (variáveis)** para tema consistente em todo o projeto
- **IntersectionObserver API** para animações de scroll reveal sem biblioteca
- **Backdrop-filter** para efeito glassmorphism na navbar
- **CSS Grid + Flexbox** para layouts responsivos complexos
- **Máscaras de input** com JavaScript puro (CPF e telefone)
- **Modais acessíveis** com suporte a fechar com `ESC` e clique fora
- **Animações CSS** com `@keyframes` e `animation-delay` para stagger effect
